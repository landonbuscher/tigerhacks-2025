# Space Tycoon

Build a profitable space tycoon with conveyers, droppers, upgraders, and furnaces.

# Features

-   Move with WASD around the map
-   Energy
    -   Solar panels generate energy.
    -   Each item placed uses a certain amount of energy per tick.
    -   Keep placing solar panels to keep up with demand.
-   Items
    -   Menu on right with categories conveyer, drills, refiners, export docks, solar panels
        -   Left/right to nav between categories
        -   Up/down to scroll between items
        -   When placing an item, the selected item in the shop menu is placed if you have enough money
    -   Sell items for 100% return
